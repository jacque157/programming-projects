Ovládanie:
Esc -> koniec hry

Pohyb myši -> natočenie kamery.
Pravé tlačidlo myši -> výmena zbrane.
Ľavé tlačidlo myši -> výstrel / čarovanie.

Pohyb je vzhľadom na natočenie kamery:
W -> pohyb smerom dopredu
S -> pohyb smerom dozadu
A -> pohyb smerom doľava
D -> pohyb smerom doprava
Shift -> pri držaní rýchly pohyb.

R -> prebytie aktívnej zbrane.
E -> zdvihnutie predmetu zo zeme

Cieľom hry je nájsť východ reprezentovaný drevenými dverami.
Hráč hru prehráva ak príde o všetko zdravie alebo zdravý rozum.
Revolver je slabší ako mágia, avšak mágia stojí zdravý rozum.

V ceste hráča stoja okultisti schopný čarovať a obrovské pavúky.

Predmety sa okamžite použíjú po ich zdvihnutí zo zeme.
